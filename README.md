# Tinder for Food

A swipe-based app to discover dishes and restaurants nearby. Built with Flutter and FastAPI.

## Features
- Swipe-based dish discovery
- Location-based filtering
- Favorites and chat with dish

## Tech Stack
- **Frontend**: Flutter
- **Backend**: FastAPI
- **Database**: PostgreSQL
